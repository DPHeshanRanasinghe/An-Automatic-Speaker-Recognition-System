# Voice Recognition System (MATLAB)

This project implements the front-end of a speaker recognition system using
classical speech signal processing techniques.

## Pipeline
Raw Speech → Preprocessing → Framing & Windowing → Feature Extraction (MFCC) → Modeling → Identification

## Structure
- src/        : Core reusable functions
- experiments/: Scripts for visualization and testing
- data/       : Recorded speech
- results/    : Generated figures

## Status
Front-end DSP pipeline completed.
